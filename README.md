# Traffic-Flow-Optimization-System
AI driven traffic flow optimization system fuses historical intelligence with real-time perception to dynamically adapt signal timings — turning every intersection into a responsive, self-optimising node.
This system is an intelligent, multi-layered traffic management pipeline that bridges **historical machine learning** with **live computer vision** to automate civic infrastructure. Instead of relying on static timers, the system dynamically adjusts traffic signal timings based on real-time vehicle detection and predicted traffic behavior.
